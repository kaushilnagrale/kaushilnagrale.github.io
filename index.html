About Me

I'm a software and data developer with 3 years of hands-on experience designing and shipping intelligent, data-driven systems across NLP, LLM applications, IoT, and computer vision. My toolkit spans the full stack of modern data and backend engineering — Node.js and Python for application and pipeline development, and a data science stack covering NumPy, Pandas, scikit-learn, TensorFlow, and PyTorch for modeling and experimentation. I work across AWS and Azure using native SDKs, and reach for Jupyter, MLflow, and HuggingFace when the work calls for it.

I've built RESTful APIs, automated data workflows, and storage layers using PostgreSQL/MySQL and NoSQL — applied across domains from real-time sensor ingestion and edge inference to LLM integrations and vision-based processing. I also did part-time web development during university, which rounded out my full-stack instincts. My undergraduate background in mechanical engineering gives me a useful systems-thinking lens. Currently wrapping up my final semester at ASU, based in Tempe, AZ.
